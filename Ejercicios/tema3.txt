Tema 3: Mover, copiar y renombrar (mv y cp)
Comandos:
mv   mover o renombrar 
cp   copiar 
cp -r  copiar carpetas 
Ejemplo guiado
Mover archivo a una carpeta 
mv nota.txt lista.txt docs/
Comprobar:
ls docs
Renombrar un archivo 
mv readme.md README.md 
Copiar un archivo
cp README.md docs/
Copiar una carpeta Completa 
cp -r docs docs_backup
